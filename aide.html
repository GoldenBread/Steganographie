Aide

Pour pouvoir utiliser ce programme, il faut commencer par connaître son but.

La stéganographie est l'art de la dissimulation. Elle consiste à cacher un message au sein d'un autre message anodin, de sorte que l'on ignore l'existence même du secret. Alors que la cryptographie repose sur le fait que le message ne soit pas compris, la stéganographie repose sur le fait que le message ne soit pas trouvé.

Dans ce programme le message que vous voulez cacher est dissimulé dans votre image.

Pour ne pas se rendre compte du subterfuge l'image doit avoir le même poids avant et après son traitement pour se faire nous vous recommandons d'utiliser une image non compressé au format .bmp.

Une fois cet élément requis en votre possession suivez les étapes suivantes :


  1. Cliquez sur parcourir pour insérer votre image

  2. Insérer votre message à cacher

  3. Si vous voulez suivre l'avancement du processus en détail ouvrez le terminal

  4. Cliquez sur le bouton Encodage pour lancer le processus Texte vers Image

  5. Une fois le processus fini « enregistrer sous » l'image à l'aide du menu contextuel (clique droit) OU cliquez sur le bouton Téléchargement



Pour faire le processus inverse, soit le décodage d'une image contenant un message :


  1. Cliquez sur parcourir pour insérer l'image codée

  2. Si vous voulez suivre l'avancement du processus en détail ouvrez le terminal

  3. Cliquez sur le bouton Décodage pour lancer le processus Image vers Texte
